HTML

  • HTML means Hyper Text Markup Language
  • The head element contains information that helps the user to render the page properly
  • The body element contains the content that is shown to the user
  • html represents the start or the root of our HTML documemnt

CSS

  • CSS stands for Cascading Style Sheets
  • Is the language used to describe the presentation of the website
  • Describes how each element of the website is rendered
  • A margin property indicates how much space we want around the outside of an alement
  • The padding property adds space around the content inside the element

GIT

  • Git bash is a programm that interacts with the computer with commands
  • Git bash replaces the GUI
  • Git hub is a cloud server that allows developers to interact with codes completely remotely
  • git status checks what branch we are currently on

Javascript

  • Is a lightweight programming language that is used to provide logic and interactivity for websites
  • Vatiables are containers that store values
  • Comments are snippets of text that can be added along with the code
